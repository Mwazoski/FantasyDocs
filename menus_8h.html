Parámetros

◆ menu_generico()

ESTADO_MENU menu_generico ( const char *  titulo,
const MenuOpcion *  opciones 
)

Genera un menú a partir de un título y una serie de opciones.

Parámetros
[in]tituloTítulo del menú.
[in]opsVector de opciones. Éste debe de acabar en NULL.
[out]retEl resultado. Éste devolverá un número del 0 al n, siendo n el número de opciones. Del 1 al n corresponden las opciones en sí. El 0 corresponde al usuario no introducciendo ninguna respuesta - a lo cual se suele tratar como que el usuario quiere salir del menú.
