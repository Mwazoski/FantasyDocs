Función para probar si el nick y la contraseña dados para acceder al sistema son válidos, en cuyo caso se le ofrecerá acceso a tal usuario.

Parámetros
[in,out]userPuntero a usuario, que apuntará al usuario pedido si procede.
[in]nickEl nick del usuario.
[in]passLa contraseña del usuario.
[out]retEl resultado. Si es un 0, procede, si es un 1, el acceso no fue permitido (en consola se escribe la razón).
